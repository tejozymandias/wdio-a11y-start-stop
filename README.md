# WebdriverIO A11y Start Stop Scan

## Quick Start

1. **Clone the repo**
   ```bash
   git clone https://github.com/tejozymandias/wdio-a11y-start-stop.git
   ```

2. **Add BrowserStack credentials to `wdio.conf.js`**
   - Update `user` and `key` with your BrowserStack username and access key

3. **Install dependencies**
   ```bash
   npm install
   ```

4. **Run tests**
   ```bash
   npm run test
   ```

> **Note:** This will run the `bstackLogin.spec.js` file located in `test/specs/bstack`
